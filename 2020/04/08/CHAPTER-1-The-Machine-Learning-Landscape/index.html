Summary

  1. about online learning

    if you have limited computing resources: once an online learning system has learned
    about new data instances, it does not need them anymore, so you can discard them.

  2. about learning rate

    it’s a kind of state transformer

    One important parameter of online learning systems is how fast they should adapt to
    changing data: this is called the learning rate. If you set a high learning rate, then your
    system will rapidly adapt to new data, but it will also tend to quickly forget the old
    data.

  3. learning by heart

    this kind of leaning is only learning by memory, just like some human’s learning, only can reproduce what it memorized.

    this kind of learning is lack abstract ability, data in ,data out.

    so In my learning progress, a pretty important aspect is abstract ability. From specification to abstract.

  4. similarity function

    This method is used in style transform, and word embedding. But they are model based learning.

  5. data && algorithm && iteration

    what we can know from this picture, just the relationship of data and algorithm, more than this is that we must notice a ting detail, the iteration.

    So what I want to talk about is generalize it to human been,there a lot of people search for a good method then they start to learn, but it is false, if we contrast human’s learning method and trained model weight. we can know even we start from scratch ,we can reach a pretty high performance finally, and in the learning process, we are continue to tweak our learning method to adapt our self, it’s a kind of self optimum.

    What caused most of people failure is the iteration, they haven’t insist on the last.
